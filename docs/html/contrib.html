Contributing

Repository

All of our code is at github. We accept pull requests and issues there. All of our code, plus our server configs are there. We generally like that the test cases pass before we accept the pull request but we're happy to have help.

Contact Info

If you have a question that doesn't fit well in an issue tracker, or just want to talk about bittol you can email the development group at bittoll-dev@googlegroups.com.
