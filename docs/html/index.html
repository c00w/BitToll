BitToll

What is BitToll?

BitToll is a different way to use bitcoins. Rather than sending transactions over the network, you request payments on a bittoll server. The server maintains balances and allows transactions to occur as fast as it can generally within 200ms. There are multiple ways to deposit bitcoins inside of the bittoll server, both mining and by sending them via the bitcoin network.

Whats the point?

Bitcoins are a very interesting currency, however they are much too slow to be used for automated transactions, such as charging for a website load, or paying 1 cent per email. Bittoll was created to resolve this problem and make it easy to use bitcoins in an automated manner.

What features does bittoll have?

Bittoll lets you deposit bitcoins into the node via a normal bitcoin transaction. It also lets you deposit into the node via mining. It lets you create payments to be fufilled, check whether they have been fufilled, and to pay an outstanding payment. It also allows you to withdraw your money into the bitcoin network.

How do I start using it?
